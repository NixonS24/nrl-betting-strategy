# Hypothesis: H_008

## ID
H_008

## Logic
Bookmaker margin intensity may vary enough across matches to create mispricing pockets.

The dataset includes bookmaker implied home and away probabilities, which makes it possible to estimate the two-way overround for each match. If higher-margin matches are less efficient, then bookmaker overround may interact with existing venue-bias signals or with simple home/away calibration.

This is a pure market-structure hypothesis using currently available fields.

## Variables
- `date`
- `season`
- `venue_name`
- `home_team`
- `away_team`
- `result`
- `bk_home_close`
- `bk_away_close`
- `bk_implied_home`
- `bk_implied_away`

## Success Metric
- Compute two-way bookmaker overround as `bk_implied_home + bk_implied_away`.
- Bucket matches by low, medium, and high overround.
- Test whether calibration error or venue-strategy ROI differs materially by overround bucket.
- Require explicit reporting of bucket definitions and sample sizes.
- Require `p_value < 0.05` for any claim that margin intensity changes predictive quality.
- Report whether excluding high-overround matches improves the base venue strategy.
- Treat any result as exploratory if the test is purely in-sample.
