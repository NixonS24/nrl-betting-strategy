# Hypothesis: H_007

## ID
H_007

## Logic
Bookmaker closing-line movement may carry an independent signal even before combining it with the existing venue strategy.

The current project already treats exchange line movement as useful market information. The local schema also contains bookmaker opening and closing prices, which allows a direct test of whether home-side price shortening or drifting between `bk_home_open` and `bk_home_close` is directionally informative and whether that signal improves when restricted to strategy venues.

This is a market-behaviour hypothesis, not a team-strength hypothesis.

## Variables
- `date`
- `season`
- `venue_name`
- `home_team`
- `away_team`
- `result`
- `bk_home_open`
- `bk_home_close`
- `bk_away_open`
- `bk_away_close`

## Success Metric
- Build opening-to-closing movement buckets for the home side using bookmaker prices.
- Test whether home shortening or drifting is associated with actual home win rate in the full bookmaker sample.
- Re-test inside known strategy venues to see whether line movement improves the existing venue-bias rule.
- Require explicit reporting of sample sizes by bucket.
- Require `p_value < 0.05` for any directional claim.
- Report ROI impact of layering a bookmaker-movement filter onto the venue strategy.
- Treat any result as exploratory if the test is purely in-sample.
