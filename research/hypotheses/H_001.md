# Hypothesis: H_001

## ID
H_001

## Logic
Home advantage in the NRL has a confirmed declining trend (p=0.015). Our current venue baselines use all-time data (1998–2025), which may over-estimate home advantage at legacy venues and under-estimate it at modern high-performance hubs (e.g., AAMI Park, BlueBet Stadium). 

Recalibrating baselines to a modern "high-fidelity" window (2019–present) should improve strategy ROI and signal reliability by reflecting the current state of home/away performance dynamics.

## Variables
- `venue_name`
- `season`
- `bk_home_close` (Bookmaker closing home odds)
- `result` (Home Win / Away Win)

## Success Metric
- ROI improvement in backtest using 2019+ baselines vs all-time baselines.
- Target p-value < 0.05 for new venue inclusions (e.g., BlueBet Stadium).
