# Hypothesis: H_009

## ID
H_009

## Logic
Market calibration may be asymmetric across the full odds curve rather than only at one narrow coin-flip bucket.

The project has already rejected the specific claim that 45–55% home prices create a robust standalone edge. A broader market hypothesis is still valid: favorites and longshots may be miscalibrated differently across bookmaker and exchange samples, especially once grouped into stable odds buckets rather than a single narrow range.

This tests a classic market-shape question using only current local fields and avoids over-committing to the previously rejected coin-flip narrative.

## Variables

- `season`

- `bk_away_close`

- `bk_implied_away`
- `bf_home_open`
- `bf_away_open`
- `bf_implied_home`
- `bf_implied_away`

## Success Metric
- Build stable home-price buckets for bookmaker and Betfair samples separately.
- Compare implied home probability against realised home win rate across the full odds curve.
- Test whether favorite or longshot buckets show persistent calibration error rather than isolated small-sample noise.
- Require explicit reporting of bucket definitions, realised win rates, implied win rates, and sample sizes.
- Require `p_value < 0.05` for any directional claim.
- Report whether any calibration asymmetry meaningfully improves venue-strategy filtering or market-side selection.
- Treat any result as exploratory if the test is purely in-sample.
