# NRL Gambling Bias Research

## Objective
This project aims to investigate and identify statistically significant biases in Australian NRL gambling markets, focusing on Draw Bias (big teams), Form/Momentum Bias, and Venue/Home Bias, and to develop a strategy to exploit these inefficiencies for profit.

## Project Structure
- `data/`: Raw and processed historical NRL data.
- `src/`: Python source code for data ingestion, analysis, and strategy.
- `notebooks/`: Jupyter notebooks for exploratory data analysis (EDA).
- `requirements.txt`: Python package requirements.

## Current Hypothesis
- **Draw Bias:** Market underrepresents the probability of draws between "big teams" (high-profile/top-tier clubs).
- **Form Bias:** Market overreacts or underreacts to recent team performance.
- **Venue Bias:** Market miscalculates home advantage at specific venues.
